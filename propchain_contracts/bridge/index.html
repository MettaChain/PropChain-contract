Module bridge 

Expand description

Cross-chain bridge types and trait definitions.

This module contains all bridge-related types, status enums, configuration structures, and trait definitions for cross-chain property token bridging.

Granular pause flags. Each operation class can be paused independently, or all_operations can be set as a master kill-switch. This lets the security team freeze e.g. only new requests while still allowing in-flight settlements to complete.

Tunable thresholds that drive automatic pausing on suspicious activity. All bounds are inclusive — hitting >= the configured value triggers.
