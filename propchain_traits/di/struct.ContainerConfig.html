Expand description

Holds all optional service AccountIds for a contract.

Embed this struct directly in your #[ink(storage)] struct. It is intentionally flat (no Mapping) so that the full config can be read in a single storage access.

All fields are Option<AccountId> — None means “not yet registered”. Use ContainerConfig::register / ContainerConfig::unregister to mutate, and ContainerConfig::resolve to look up at call time.
